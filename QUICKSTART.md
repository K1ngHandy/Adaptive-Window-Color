# Quick Start Guide

Get started with Adaptive Window Color in just a few minutes!

## Installation

### Step 1: Get the Code
```bash
git clone https://github.com/K1ngHandy/Adaptive-Window-Color.git
cd Adaptive-Window-Color
```

### Step 2: Install & Build
```bash
npm install
npm run build
```

### Step 3: Load in Chrome
1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select the `dist` folder

### Step 4: You're Ready!
The extension is now active. Visit any colorful website to see it in action!

## First Use

1. **Pin the Extension** (optional but recommended)
   - Click the puzzle piece icon in Chrome toolbar
   - Find "Adaptive Window Color"
   - Click the pin icon

2. **Try It Out**
   - Visit https://www.stripe.com (blue theme)
   - Visit https://www.spotify.com (green theme)
   - Visit https://www.netflix.com (red theme)
   
   Watch your browser theme adapt automatically!

3. **Control It**
   - Click the extension icon to open settings
   - Toggle on/off as needed
   - Theme updates automatically when enabled

## What You'll See

### Theme Changes Include:
- **Tab bar color** - Matches dominant page color
- **Window frame** - Subtle color variation

## Tips

### For Best Experience:
- Visit sites with strong visual branding
- Allow 1-2 seconds for theme to update after page load
- Works best on sites with colorful images or backgrounds

### Troubleshooting:
- **Theme not updating?** Refresh the page
- **Colors look off?** Some sites have limited color content
- **Need default theme back?** Just toggle off in the popup
